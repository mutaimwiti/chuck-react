# Chuck React

### Setup
#### Clone this repository
Use one of the two commands:
- `git clone https://github.com/mutaimwiti/chuck-react.git` (HTTPS)
- `git clone git@github.com:mutaimwiti/chuck-react.git` (SSH)

#### Go to project
- `cd chuck-react`

#### Install dependencies

- `yarn`

### Start application

- `yarn start`

### Run tests

- `yarn test`
